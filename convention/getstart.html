Web前端入门指南

前端的基础知识、技术与技能

前言




初级

- 知识篇 -

1. W3school

网址：http://www.w3school.com.cn/

参考时间：20 小时

要求：完整看一遍。要特别留意定义部分，要做每个教程后的测试并记录分数

必看内容

  • HTML, HTML5

  • CSS, CSS3

  • Javascript, DOM, AJAX, JSON

选看内容

  • XML, PHP, 其他

2. CSS手册

网址：http://css.doyoe.com/

参考时间：10 小时

要求：完整遍历，包括每个链接

3. 红宝书

即《Javascript高级程序设计3》

参考时间：70 小时

要求：完整看一遍

正则表达式

网址：http://regexr.com/

[推荐文章]

  • Javascript诞生记

  • JavaScript运行机制详解

  • 学习Javascript闭包

[课外知识]

  • 浏览器和网络

  • 网络的演变

  • 浏览器的工作原理

- 技术篇 -

1. jQuery

版本：2.0+

文档：http://api.jquery.com/

中文版：http://www.jquery123.com/

整理版：http://www.css88.com/jqapi-1.9/

参考时间：10 小时

要求：将其中一个文档版本完整看一遍

2. Bootstrap

版本：3.3.5+

文档：http://getbootstrap.com/

中文版：http://www.bootcss.com/

要求：将其中一个文档版本完整看一遍

3. doT.js

文档：http://olado.github.io/doT/

要求：能写实例

- 技能篇 -

1. Sublime Text

网址：http://www.sublimetext.com/

要求：了解所有的操作，并亲自操作一遍

Emmet

文档：http://docs.emmet.io/

要求：熟练使用

2. Chrome Devtools

文档：https://developer.chrome.com/devtools/index

要求：遍历文档，了解所有的操作，并亲自操作一遍，其中要特别注重 Debug 功能

Debug

文档：https://developer.chrome.com/devtools/docs/javascript-debugging

要求：能够独立进行代码分析

3. Shortcuts

要求：尽可能地使用键盘取代鼠标

  • Windows 快捷键

  • Chrome 快捷键

  • Sublime Text 快捷键
